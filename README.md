# elm-planning-poker

Planning poker app. Frontend is in Elm, Backend is in Crystal (using the Kemal framework).
Inspired by [crystal-elm-chat](https://github.com/martimatix/crystal-elm-chat).

## Installation

```
$ ./build.sh
```

## Usage

```
$ ./run.sh
```
